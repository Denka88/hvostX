# HvostX — Интернет-магазин товаров для домашних животных

**HvostX** — это полнофункциональный интернет-магазин для продажи товаров для домашних животных, разработанный на PHP с использованием MySQL и Bootstrap.

## 📋 Возможности

### Для пользователей

- **Каталог товаров**
  - Просмотр всех товаров с пагинацией
  - Фильтрация по категориям
  - Поиск по названию и описанию
  - Сортировка по рейтингу и популярности

- **Корзина и заказы**
  - Добавление товаров в корзину
  - Оформление заказа с выбором доставки и оплаты
  - Отслеживание статуса заказов
  - История заказов

- **Избранное**
  - Добавление товаров в список избранного
  - Быстрый доступ к любимым товарам

- **Отзывы и рейтинги**
  - Оценка товаров (1-5 звёзд)
  - Написание отзывов с модерацией
  - Защита от спама (reCAPTCHA)

- **Личный кабинет**
  - Регистрация и авторизация
  - Редактирование профиля
  - Управление аватаром
  - Просмотр истории заказов

- **Дополнительно**
  - Страница новостей компании
  - Страница партнёров
  - Форма обратной связи
  - Адаптивный дизайн (Bootstrap 5)

### Для администраторов

- **Панель управления**
  - Дашборд со статистикой
  - Управление пользователями
  - Управление товарами (CRUD)
  - Управление категориями
  - Управление заказами (смена статусов)
  - Управление новостями
  - Управление партнёрами
  - Модерация отзывов
  - Просмотр сообщений от клиентов
  - Управление фильтрами товаров

- **Ролевая модель**
  - Разделение прав (администратор / модератор)

## 🚀 Первый запуск

### Требования

- PHP 8.0 или выше
- MySQL 5.7+ / MariaDB 10.4+
- Веб-сервер (Apache)
- Расширения PHP: `mysqli`, `session`, `gd`

### Установка

1. **Клонируйте или скопируйте файлы проекта** в дирекорию веб-сервера:
   ```
   e:\programs\xampp\htdocs\work\hvostx
   ```

2. **Настройте базу данных:**
   
   - Откройте phpMyAdmin (обычно `http://localhost/phpmyadmin`)
   - Создайте новую базу данных с именем `hvostx`:
     ```sql
     CREATE DATABASE hvostx CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
     ```
   - Импортируйте SQL-дамп из файла `assets/hvostx.sql`:
     ```bash
     mysql -u root -p hvostx < assets/hvostx.sql
     ```
     Или через интерфейс phpMyAdmin: выберите БД → Импорт → выберите файл

3. **Настройте подключение к БД:**
   
   Откройте файл `includes/db.php` и при необходимости измените параметры:
   ```php
   $servername = "localhost";
   $username = "root";
   $password = "";
   $dbname = "hvostx";
   ```

4. **Настройте reCAPTCHA:**
   
   Откройте `includes/recaptcha.php` и укажите ваши ключи:
   ```php
   define('RECAPTCHA_SITE_KEY', 'ваш_ключ_сайта');
   define('RECAPTCHA_SECRET_KEY', 'ваш_секретный_ключ');
   ```

5. **Запустите веб-сервер:**
   
   Для XAMPP:
   - Запустите Apache и MySQL из панели управления XAMPP
   - Откройте в браузере

## 🔧 Конфигурация

### Переменные сессии

```php
$_SESSION['user_id']      // ID пользователя
$_SESSION['user_name']    // Имя пользователя
$_SESSION['user_login']   // Логин
$_SESSION['user_email']   // Email
$_SESSION['user_role']    // Роль (admin/moderator)
$_SESSION['user_avatar']  // Аватар
$_SESSION['cart']         // Корзина (массив)
```

### Статусы заказов

- `pending` — Ожидает обработки
- `processing` — В обработке
- `completed` — Выполнен
- `cancelled` — Отменён

## 🔐 Безопасность

- Пароли хешируются через `password_hash()`
- Prepared statements для защиты от SQL-инъекций
- reCAPTCHA для защиты от спама
- Проверка сессий для авторизованных действий
- Разделение прав доступа (администратор/модератор)
